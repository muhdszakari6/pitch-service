
# Soccer Pitch Booking API

A REST API for managing soccer pitch bookings, built with Spring Boot.

## Features

- User registration with email verification
- JWT-based authentication
- Soccer pitch creation and management
- Session-based booking slots
- Booking management with status tracking (PENDING / CONFIRMED / CANCELLED)
- Role-based access (USER / PARTNER / SUPER_ADMIN)
- Password reset flow

## Tech Stack

- Java 17
- Spring Boot 3.4.1
- Spring Security + JWT
- MySQL
- Gradle
